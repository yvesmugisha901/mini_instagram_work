# Mini Social Network Login System (Instagram Inspired)

## Overview

This project is a simplified Instagram-style login system built using
PHP. It includes sign-up, sign-in, validation, mock Google login, and
redirects to a CRUD system.

## Features

-   Instagram-inspired UI design\
-   Sign Up with full validation\
-   Sign In using username/email and password\
-   Mock Google Login button\
-   Redirects to existing CRUD project\
-   Error handling for all login/sign-up mistakes

## Pages Included

-   Sign Up Page\
-   Sign In Page\
-   Dashboard Page\
-   CRUD page (redirect target)

## Technologies Used

-   PHP\
-   HTML/CSS\
-   MySQL\
-   Mock Google login simulation
